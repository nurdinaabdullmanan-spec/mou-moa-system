# 5. Creating Database Tables

### Sample Coding

```python
cursor.execute("""
CREATE TABLE IF NOT EXISTS users (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    username TEXT,
    email TEXT,
    password TEXT
)
""")
```

### Description

This section creates the database tables required by the system.

The `users` table stores user account information such as username, email, and password for authentication purposes.

The `collaboration_data` table stores all MoU/MoA agreement information including:

* Agreement title
* Duration
* Department
* Partner institution
* Country
* Agreement category

The `AUTOINCREMENT` function automatically generates unique IDs for each record.

### Description

The `session_state` feature stores user login status temporarily while the application is running. If the user successfully logs in, the system saves the session data and allows access to the dashboard without requiring repeated login actions.
